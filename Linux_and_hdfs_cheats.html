MEMORY RELATED OPERATIONS

COMMAND OPERATION
$df -h Check the memory alloted for all the volumes
$lsblk Check details for all the partitions
$vgdisplay Indicates the free physical extents(the extent by which a logical volume can be extended)
$vgs -o +vg_free_count,vg_extent_count Extension of the above vgdisplay -- shows in more human reaadable format
$lvresize -L +300GB /dev/mapper/vg_master1-lv_homeIncreases the size of home volume by 300GB
$lvextend -L 700G /dev/mapper/vg_master1-lv_root Increases the size of root volume to 700GB
Example:

In case there are no extents(i.e. memory) available to extend a logical volume, some other volume can be shrunk and the additional space thus created can be used to increase the size of any other logical volume
COMMAND OPERATION
$lvresize -L -30GB /dev/mapper/vg_webserver-lv_home Decreases the size of "home" by 30GB
$lvresize -L +30GB /dev/mapper/vg_webserver-lv_root Adds the 30GB that wass deallocated in the above step to the "root"

DIRECTORY(DETAILS AND TRAVERSAL) RELATED OPERATIONS

COMMAND OPERATION
$pwd Display the current working directory
$mkdir directory_name Ceates a new directory
$cd directory_name Changes the current working directory to the directoy name specifed(downwards traversal)
$cd .. Changes the current working directory to a level up(upwards traversal)
$cd ~ Changes the current working directory to root(upwards traversal)
$ls Display the folders and the files present in the current working directory
$ls -l Display the file permissions
$ls -a Display all the files(including the hidden ones)
$du -sh Displays the memory used by the current folder
$du -h Displays the memory used by the current folder, its subfolder and all the files under it
$rmdir directory_name Deletes the directory specified by the directoy name(the directory should be empty before deletion)
$rm -rf directory_name Deletes the directory specified by the directoy name and all the files under that directory

FILE OPERATIONS(READ, WRITE, COPY, DELETE, etc.)

COMMAND OPERATION
$vi filename Creates a file as denoted by the filename and opens it for editing in VI editor
$rm filename Removes the file specified by the filename(Also used to move files from one directory to other)
$mv filename1 filename2 Renames filename1 to filename2
$cp filename1 filename2 Copies the contents of the filename1 to a new file named filename2

RUNNING AND SCHEDULING JOBS

COMMAND OPERATION
$./my-shell-script.sh & Run a script in the background
$nohup ./my-shell-script.sh & Run the script even after logging the user out
$at -f my-shell-script.sh 10 am tomorrowSchedule a script to run at the specified time

NETWORK RELATED DETAILS AND OPERATIONS

COMMAND OPERATION
